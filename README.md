# 💰 Controle Financeiro Familiar

Dashboard completo de controle financeiro com Google Sheets + análise inteligente.

## 🎯 O que tem no produto

✅ Planilha estruturada no Google Sheets
✅ Dashboard com 5 páginas de análise
✅ Alertas personalizados
✅ Dicas baseadas nos seus dados
✅ Funciona em celular e computador

## 📱 Como usar (2 passos)

### 1. Configure a planilha

1. Importe `planilha_financeira_familiar.xlsx` no Google Sheets
2. Compartilhe como "Qualquer pessoa com o link pode VER"
3. Copie o link

### 2. Conecte no dashboard

1. Abra o dashboard
2. Cole o link da planilha
3. Pronto!

*O dashboard salva no navegador. Você configura 1× e nunca mais precisa.*

## 📊 Páginas do dashboard

**📊 Visão Geral** — Saldo, gráficos, alertas  
**🔍 Análise** — Evolução mensal, maiores gastos  
**🎯 Orçamento** — Planejado vs realizado  
**📈 Projeção** — Vai sobrar ou faltar?  
**💡 Dicas** — Personalizadas pros seus dados  

## ❓ Dúvidas

**É de graça?** Sim, sem mensalidade.  
**É seguro?** Seus dados ficam na sua conta Google.  
**Funciona no celular?** Sim!  
**Preciso instalar algo?** Não.
